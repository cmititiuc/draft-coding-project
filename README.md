# README

## seed database

    rails db:setup

## run tests

    rails db:test:prepare
    rails test

## start rails server

    rails s

## get a single player

    http://localhost:3000/players/:id

## get all players by sport

    http://localhost:3000/players/football
    http://localhost:3000/players/basketball
    http://localhost:3000/players/baseball
